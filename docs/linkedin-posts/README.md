# LinkedIn Post Drafts

These drafts position the AskOC AI Concierge repo for hiring managers, CTOs, and technical leaders.

The writing style follows the practical lessons in [docs/eos.md](../eos.md):

- use active voice,
- prefer concrete details,
- keep one thought per paragraph,
- omit needless words,
- place the strongest point at the end.

The tone is casual and technical: playing with a useful idea, building a real slice, and using AI as an assistive tool. Use short paragraphs, but avoid stacking too many single-sentence paragraphs. Do not use em dashes. The posts do not claim real student data, live enterprise integrations, automated approvals, or production deployment.
